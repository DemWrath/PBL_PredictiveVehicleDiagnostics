Predictive Vehicle Diagnostics Using Deep Learning

AI-powered proactive engine fault detection using BiLSTM with Attention on OBD-II telemetry, deployed as a production REST API.

Problem Statement

Modern vehicles generate continuous OBD-II telemetry but consumer systems only provide reactive fault codes. This project builds a proactive AI system that detects engine anomalies before failure using deep learning on time-series OBD-II data.

Innovation

Unsupervised autoencoder-based anomaly detection eliminates need for labeled faults. BiLSTM with attention captures temporal dependencies across 1200 timesteps.

Production Ready

Deployed as Dockerized Flask REST API on Render with threshold optimization for recall-critical automotive applications.

System Architecture

Phase 1: Preprocessing (1,699 sequences of 1200 timesteps, 10 sensors)
Phase 2: Autoencoder training for anomaly detection
Phase 3: Reconstruction-based labeling (1,334 normal / 365 fault)
Phase 4: BiLSTM Classifier with Attention mechanism
Phase 5: Threshold optimization (0.52, F2-optimized)
Phase 6: Dockerized deployment ‚Üí Flask API ‚Üí Render cloud

Results & Performance

Recall 90.5%
Precision 38.3%
Threshold 0.52
